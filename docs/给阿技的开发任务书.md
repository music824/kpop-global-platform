# KPOP 全球平台 - 开发任务书
## 给阿技的技术开发任务

---

## 📋 项目背景

我们正在开发一个 **AI驱动的全球KPOP实时活动与数据平台**。

不是普通新闻站，是：
- 全球KPOP活动数据库
- AI自动更新内容
- 多语言支持
- 未来可扩展APP/会员系统

**技术栈：**
- 前端：Next.js 15 + TypeScript + TailwindCSS
- 后端：Supabase (PostgreSQL)
- 部署：Vercel
- AI：自动抓取 + 翻译系统

1. 创建 GitHub 仓库（命名为 `kpop-global-platform`）
2. 用 Next.js 15 初始化项目：
   ```bash
   npx create-next-app@latest kpop-global-platform --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"
   ```

3. 安装依赖：
   ```bash
   npm install @supabase/supabase-js @supabase/ssr swr date-fns lucide-react
   ```

### 任务2：创建基础组件

1. **布局组件**（`src/components/layout/`）：
   - `Header.tsx` - 顶部导航栏
   - `Footer.tsx` - 底部
   - `Sidebar.tsx` - 侧边栏（可选）

2. **首页组件**（`src/components/home/`）：
   - `HeroBanner.tsx` - 大横幅
   - `FlashNewsTicker.tsx` - 快讯滚动条
   - `HotArtists.tsx` - 热门艺人卡片

---

### 任务3：创建页面

1. **首页**（`src/app/page.tsx`）：
   - Hero Banner（今日热门活动）
   - 快讯滚动（AI生成的实时快讯）
   - 热门艺人（Grid展示）
   - 最新新闻列表

2. **活动页面**（`src/app/events/page.tsx`）：
   - 活动列表（筛选：国家/类型/时间）
   - 日历视图

3. **艺人页面**（`src/app/artists/page.tsx`）：
   - 艺人列表（按热度排序）
   - 艺人详情页（`src/app/artists/[slug]/page.tsx`）

4. **新闻页面**（`src/app/news/page.tsx`）：
   - 新闻列表（筛选：类型/艺人）
   - 新闻详情页

5. **地图页面**（`src/app/map/page.tsx`）：
   - 全球活动地图（可用 Leaflet 或 Mapbox）

---

### 任务4：连接Supabase数据

1. 创建 Supabase 客户端（`src/lib/supabase/client.ts`）
2. 创建 API 层（`src/app/api/`）
3. 用 SWR 或 React Query 获取数据
4. 实现 SSR（服务端渲染）优化SEO

---

### 任务5：设计风格

- **深色科技感**：黑色背景 + 银色文字
- **KPOP活力点缀**：渐变紫/粉（#8B5CF6 → #EC4899）
- **参考**：Apple官网 + HYBE官网风格
- **字体**：无衬线，干净利落
