> 基于 Next.js 15 + TypeScript + TailwindCSS + shadcn/ui + Supabase

## 1. 现有组件与 Supabase 表对应关系

| 组件 | 数据需求 | 对应 Supabase 表 |
|------|---------|-----------------|
| `HeroBanner` | 焦点演唱会/活动 | `events` |
| `FlashNewsTicker` | 快讯列表 | `news` (type='flash') |
| `HotArtists` | 艺人列表 + 热度分 | `artists` |
| `LatestNews` | 最新新闻 | `news` |
| `news/page.tsx` | 新闻列表 + 筛选 | `news` |
| `news/[id]/page.tsx` | 新闻详情 + 相关艺人 | `news` + `artists` |

### 预计数据表设计（参考）

```sql
-- events 表
id, title, subtitle, artist_id, artist_name, country, city, venue, 
date, time, status, poster_image, created_at

-- artists 表
id, name, name_en, group_name, profile_image, hot_score, country, created_at

-- news 表
id, title, summary, content, image_url, source, publish_date, 
news_type, tags[], artist_ids[], created_at
```

## 3. SWR 调用基本写法示例

### 初始化 SWR Client

```typescript
// src/lib/supabase/swr-client.ts
import { createClient } from '@/lib/supabase/client'

export const supabase = createClient()
```

### 基础查询模式（带 SWR 缓存）

```typescript
// 示例：获取热门艺人
import useSWR from 'swr'
import { supabase } from '@/lib/supabase/swr-client'

const fetcher = async (query: any) => {
  const { data, error } = await query
  if (error) throw error
  return data
}

// 在组件中使用
function HotArtists() {
  const { data: artists, error, isLoading } = useSWR(
    [supabase.from('artists').select('*').order('hot_score', { ascending: false }).limit(6)],
    ([query]) => fetcher(query)
  )

  if (isLoading) return <Skeleton />
  if (error) return <div>加载失败</div>

  return (
    <div className="grid grid-cols-6 gap-4">
      {artists?.map((artist) => (
        <ArtistCard key={artist.id} artist={artist} />
      ))}
    </div>
  )
}
```

### 带筛选条件的查询

```typescript
// 示例：带筛选的新闻列表
import useSWR from 'swr'
import { supabase } from '@/lib/supabase/swr-client'

function NewsList({ newsType }: { newsType: string }) {
  const query = newsType === 'all'
    ? supabase.from('news').select('*').order('created_at', { ascending: false })
    : supabase.from('news').select('*').eq('news_type', newsType).order('created_at', { ascending: false })

  const { data: news, isLoading } = useSWR(
    ['news-list', newsType],
    () => fetcher(query)
  )

  // ...
}
```

### 新闻详情页

```typescript
// 示例：新闻详情
function NewsDetail({ newsId }: { newsId: string }) {
  const { data: news, isLoading } = useSWR(
    ['news-detail', newsId],
    () => fetcher(supabase.from('news').select('*').eq('id', newsId).single())
  )

  const { data: relatedArtists } = useSWR(
    news?.artist_ids ? ['related-artists', news.artist_ids] : null,
    () => fetcher(supabase.from('artists').select('*').in('id', news.artist_ids))
  )

  // ...
}
```

## 4. 注意事项

1. **环境变量**：确保 `.env.local` 配置了 `NEXT_PUBLIC_SUPABASE_URL` 和 `NEXT_PUBLIC_SUPABASE_ANON_KEY`
2. **SWR 缓存 Key**：使用稳定的 key 数组，避免重渲染
3. **空数据处理**：每个组件需处理 `isLoading` 和 `error` 状态
4. **实时订阅**（可选）：可用 `supabase.channel` 实现新闻/艺人热度的实时更新
